# 项目简介 python867

本项目是一款基于Python和Web的聊天机器人系统，使用Django框架和MySQL数据库，采用前后端分离的方式进行开发。

## 技术栈

- 架构：B/S、MVC
- 开发语言：Python
- 框架：Django
- 数据库：MySQL

## 功能模块

### 系统角色

系统设计两个角色：普通用户和管理员。

### 前台系统

- **登录注册**：用户可以进行注册和登录，注册后可使用系统功能。
- **首页展示**：展示系统的功能入口和基本信息。
- **个人信息**：用户可以查看和修改个人信息。
- **在线聊天**：用户可以与聊天机器人进行在线聊天。
- **问答列表**：用户可以查看聊天记录和问答内容。
- **改变主题**：用户可以切换系统主题和字体。

### 后台管理系统

- **管理员端**
  - **用户信息管理**：管理员可以管理用户信息，包括添加、修改和删除用户。
  - **问答列表管理**：管理员可以查看和管理用户与聊天机器人的问答记录。
  - **系统信息管理**：管理员可以管理系统信息，如公告和新闻。

## 运行环境

- 系统环境：Windows
- 开发环境：PyCharm、Python 3.x、MySQL 5.x
